    • The Stepchild is designed to be easy to assemble and build yourself without using a kit, but here are a few things to know about building the Stepchild from scratch:

  • Two of the PCBs (the main board and step button board) require tight SMD component soldering. I really recommend having these boards populated using a PCBA service! If you're using a service to fabricate your own PCBs, you should have these two fully populated with their SMD components.
  • There is a lot of soldering required to build the Stepchild both from a kit and from scratch. Most of the assembly process isn't difficult, but if you're totally new to soldering, buying a kit/scratch building might not be a great experience. Check out the build video and see if this is a project you'd feel comfortable with.

    • Tools you'll need:

  • A soldering iron + solder + be safe! Don't solder without good ventilation, and use good safety procedures if you're working with leaded solder.
  • Small clippers for trimming pins

    • And that's it!

    Unless you're printing your own case, in which case you need:

  • An FDM 3D printer with an 0.2mm nozzle, or an SLA printer -- no graphics, though! Get out your markers :)

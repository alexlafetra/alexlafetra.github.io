PREORDER

The Stepchild is a portable step sequencer, MIDI utility, and generative MIDI instrument built for experimental sequencing with hardware samplers and synthesizers.

This project started because I wanted an affordable, powerful sequencer that would let me experiment with writing MIDI the way a DAW does. The Stepchild is supposed to be a flexible, experimental, and

The Stepchild is an open source project, meaning both the software and hardware designs are accessible to everyone. You can build your own or buy a completed stepchild or kit.

The Stepchild is designed to be a:

  • MIDI interface for connecting a DAW, TRS MIDI, and DIN MIDI devices together
  • Portable live performance sequencer, with live looping and an onboard filesystem for storing sequences
  • Composition tool for experimenting with MIDI and hardware instruments
  • Generative sequencer for creating ambient samples using generative algorithms

    • The Stepchild hardware has:

  • 4 MIDI outputs, 1 hardware input, each with dual TRS/DIN inputs
  • 1 MIDI USB in/out port for connecting to a DAW or USB host device. The Stepchild can't act as a USB host... yet!
  • 4 CV output jacks to interface with modular synth systems, supporting a 0<->+12V voltage range.
  • 16 LED step buttons for old-school step sequencing and an additional input.
  • 2 rotary encoders and 1 joystick for expressive MIDI control.

    • The Stepchild software has:

  • an automation curve editor for messing with hardware instruments in real time
  • note FX for experimental songwriting
  • generative midi applications for using the Stepchild as a live instrument, including a Live MIDI Looper
  • a flexible clock for swung sequences and weird polyrhythms

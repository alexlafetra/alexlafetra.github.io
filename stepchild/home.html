Preorder the Stepchild here !

The Stepchild is a portable step-sequencer, MIDI interface, and generative MIDI instrument.

The online manual for the Stepchild can be read here; it's available to download along with the source code and some deeper documentation on the Github Page.

It's the Step-child because it's a Step-sequence, and because it's your baby.

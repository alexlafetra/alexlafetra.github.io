The Stepchild is a portable step-sequencer, MIDI utility, and generative MIDI instrument.

You can read more about the Stepchild project on the github page and find the online user manual here.

The Stepchild has:

  • four hardware MIDI inputs
  • one USB in/out port for connecting to a DAW or USB host device.
  • Three CV outputs to connect to Eurorack/CV modules

    • Each MIDI port can use both a standard MIDI DIN input jack and 3.5mm TRS jack, letting you use the Stepchild to translate between TRS, DIN MIDI, and USB MIDI signals.

    The Stepchild is designed around a graphical UI displaying its sequencer, with contains up to 127 separate tracks. Additionally, there's:

  • an automation curve editor for messing with hardware instruments in real time
  • note FX for experimental songwriting
  • generative midi applications for using the Stepchild as a live instrument, including a Live MIDI Looper
  • a flexible clock for swung sequences and weird polyrhythms
